‚öîÔ∏è EQUIPMENT SYSTEM
===========================================

EQUIPPING: /equip <item_name>
UNEQUIPPING: /unequip <item_name>

Equipment provides PERMANENT bonuses
while equipped. Only unequip to switch items.

EQUIPMENT SLOTS:
Head - Goggles, helmets, etc.
Feet - Shoes, boots
Back - Jetpacks, cloaks
Accessory - Charms, injectors
Tool - Devices, kits, cards

You can equip ONE item per slot (5 total).

EXAMPLES:

Example 1: Equipping Gear
Inventory: Night Vision Goggles (unequipped)
Command: /equip Night Vision Goggles
Result: Goggles equipped to Head slot
Bonus: -15% detection (active!)

Example 2: Switching Items
Currently equipped: Silent Shoes
Want to equip: Turbo Boots
Command: /equip Turbo Boots
Result: Silent Shoes ‚Üí inventory, Turbo Boots ‚Üí Feet slot

Example 3: Full Loadout
/equip Night Vision Goggles (Head)
/equip Silent Shoes (Feet)
/equip Jetpack (Back)
/equip Adrenaline Injector (Accessory)
/equip Hacking Kit (Tool)

Total bonuses:
- 15% detection reduction
- 20% noise reduction
- 25% escape speed
- 15% progress boost
- 25% tech success

Example 4: Unequipping
Command: /unequip Hacking Kit
Result: Hacking Kit ‚Üí inventory
Tool slot: Empty
Bonus: -25% tech success (removed)
