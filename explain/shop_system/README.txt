üìã HOW TO USE ITEMS
===========================================

BUYING ITEMS:
1. Use /shop command
2. Select category from dropdown
3. View items with prices
4. Click "Purchase" button
5. Confirm purchase

USING CONSUMABLES:
1. Use /useitem command
2. Select item from dropdown
3. Item effect activates
4. Use for next applicable action

EQUIPPING GEAR:
1. Use /equip command
2. Type item name
3. Item goes to appropriate slot
4. Bonuses apply immediately

UNEQUIPPING:
1. Use /unequip command
2. Type item name
3. Returns to inventory
4. Bonuses removed

VIEWING INVENTORY:
1. Use /inventory command
2. Filter by category (optional)
3. See all owned items
4. Check equipped gear

OPENING CHESTS:
1. Use /openchest command
2. Type chest rarity (common, rare, etc.)
3. Reveal rewards
4. Watch for mimics!
